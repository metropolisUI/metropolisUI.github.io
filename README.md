# metropolisUI


## Setup

- Clone repo: `git clone https://github.com/metropolisUI/metropolisUI.github.io`
- NPM Install: `npm install`
- Bower Install: `bower install`
- Run local server: `npm run serve`


## Deploy

Deploy in one command: `npm run deploy`
